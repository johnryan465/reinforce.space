From wikipedia:

$$P_{out} = K_{p} e(t) + p_0$$

- $p_0$: Controller output with $0$ error
- $P_{out}$: Output of the proportional controller
- $K_p$: Proportional gain
- $e(t)$: Instantaneous process error at time $t$
- $SP$: Set point
- $PV$: Process variable
